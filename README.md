**Testes de API com Cypress**

Projeto desenvolvido durante aulas práticas para aprender testes automatizados de API utilizando Cypress.
O objetivo foi praticar operações básicas de API, uso de autenticação por token e aplicação de DDT 
(Data Driven Testing) para execução de testes com diferentes conjuntos de dados.

**Conteúdos Praticados**

Durante as aulas foram abordados os seguintes tópicos:
- Primeiros passos com Cypress
- Requisições POST, PUT e DELETE
- Uso de Token de autenticação
- Implementação de DDT (Data Driven Testing)
- Organização de testes automatizados

**Tecnologias Utilizadas**
- Cypress
- JavaScript
- Node.js
- API pública: https://-petstore.swagger.io
